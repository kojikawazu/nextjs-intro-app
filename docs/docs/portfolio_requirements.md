# ポートフォリオサイト要件定義書

## 1. プロジェクト概要

### 1.1 プロジェクト名

**TechProfile Pro - フリーランスエンジニアポートフォリオサイト**

### 1.2 目的

- フリーランスエンジニアの技術スキル・経歴・実績を効果的に訴求
- クライアントからの問い合わせ獲得とビジネス機会の創出
- 専門性の高いランディングページとしての機能提供

### 1.3 ターゲットユーザー

- エンジニア採用を検討している企業担当者
- システム開発を外注検討している企業・個人
- 技術パートナーを探している事業者
- 同業エンジニア・技術コミュニティメンバー

## 2. サイト構成・機能要件

### 2.1 全体構成

**シングルページアプリケーション（SPA）**

- ナビゲーション
- ヒーローセクション
- About（自己紹介）
- Career（経歴）
- Skills（技術スキル）
- Contact（お問い合わせ）
- フッター

### 2.2 各セクション詳細要件

#### 2.2.1 ナビゲーション

- サイトタイトル：「TechProfile Pro」
- スムーススクロール対応メニュー
    - About
    - Career
    - Skills
    - Contact
- レスポンシブ対応（モバイルハンバーガーメニュー）

#### 2.2.2 ヒーローセクション

- インパクトのあるメインビジュアル
- 画像URL：提供済みUnsplash画像
- キャッチコピー・タイトル表示
- CTAボタン（Contactセクションへ誘導）

#### 2.2.3 Aboutセクション

- **プロフィール情報**

    - 名前：フリーランスエンジニア
    - プロフィール写真：提供済みUnsplash画像
    - 自己紹介文（5段落の詳細な紹介文を表示）

- **SNSリンク**
    - X（Twitter）
    - GitHub
    - Zenn
    - Qiita
    - 各SNSアイコン表示・外部リンク対応

#### 2.2.4 Careerセクション

- **経歴一覧表示**（6案件）

    1. PC基盤の統合管理システムのコンテナ化（2024年10月〜現在）
    2. PC基盤の統合管理システムの機能変更（2021年5月〜現在）
    3. 新CIサーバのリプレイス開発（2021年1月〜4月）
    4. 電力監視システムの新規開発（2019年9月〜2020年9月）
    5. 再エネ管理システムのプロトタイプ開発（2019年7月〜2020年3月）
    6. 九州のネット回線・携帯回線利用可能拡大案件（2015年4月〜2018年8月）

- **各案件表示項目**
    - 案件タイトル
    - 期間
    - チーム規模
    - プロジェクト内容
    - 技術スタック（タグ表示）
    - プロジェクトフェーズ
    - チーム内での役割

#### 2.2.5 Skillsセクション

- **技術スキル一覧**（39技術）
- **カード形式での表示**

    - 技術アイコン
    - 技術名
    - 習熟度・使用経験説明

- **主要技術カテゴリ**
    - プログラミング言語：Java、TypeScript、C/C++、Python、Go、Rustなど
    - フレームワーク：SpringBoot、Next.js、React、Django、Echo、Honoなど
    - インフラ・クラウド：AWS、GCP、Docker、Terraformなど
    - OS・ミドルウェア：Linux系OS、Tomcat、PostgreSQL、MySQLなど
    - 開発ツール：Git、GitHub Actions、Jenkins、JIRAなど

#### 2.2.6 Contactセクション

- **お問い合わせフォーム**

    - お名前（必須）
    - メールアドレス（必須）
    - お問い合わせ内容（必須）
    - 送信ボタン：「上記内容で送信する」

- **フォーム機能**
    - バリデーション機能
    - 送信確認・完了メッセージ
    - メール送信機能（実装検討）

#### 2.2.7 フッター

- コピーライト：「@techprofilepro」

## 3. 非機能要件

### 3.1 パフォーマンス

- ページ読み込み速度：3秒以内
- 画像最適化対応
- レスポンシブデザイン（スマートフォン・タブレット・PC対応）

### 3.2 SEO・アクセシビリティ

- メタタグ設定
- OGP設定
- セマンティックHTML
- キーボードナビゲーション対応

### 3.3 ブラウザ対応

- Chrome、Firefox、Safari、Edge最新版対応
- モバイルブラウザ対応

## 4. 技術仕様

### 4.1 推奨技術スタック

- **フロントエンド**：Next.js + TypeScript
- **スタイリング**：Tailwind CSS
- **デプロイ**：Vercel
- **画像最適化**：Next.js Image Optimization

### 4.2 データ管理

- **データソース**: Google Cloud Storage上のsample.json
- **アクセス方法**: GCS Public URL または Cloud Functions経由
- **キャッシュ戦略**: ビルド時取得 + ISR（Incremental Static Regeneration）

## 5. 制作スケジュール目安

- 各セクション実装

### Phase 3：テスト・最適化（1-2日）

- 動作テスト

- SEO対策

### Phase 4：デプロイ・公開（1日）

- 本番環境デプロイ
- 最終確認
